OOP Project Details

Project Overview

This Student Management System demonstrates core Object-Oriented Programming concepts implemented in C++:

  • Classes & Objects: Student, Teacher, Admin, User base class
  • Inheritance: Hierarchical inheritance with User as base class
  • Polymorphism: Virtual functions and function overloading
  • Encapsulation: Private data members with public interfaces
  • Abstraction: Abstract User class as blueprint
  • Templates: Generic printList function
  • Exception Handling: Input validation with try-catch
  • File Handling: CSV data persistence
  • Diamond Problem: Solved using virtual inheritance
  • Static Members: System-wide user tracking

Code Structure

// Abstract User Class
class User {
protected:
    int id;
    string name;
public:
    virtual void menu() = 0; // Pure virtual
    virtual ~User() {}
};

// Student inherits from User
class Student : public User {
    vector<int> marks;
public:
    void registerCourse();
    void viewGrades();
    double cgpa();
    void menu() override;
};
